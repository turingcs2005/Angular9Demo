Submitted data are logged to the console. (press F12 on Google Chrome to toggle browser console)
  Form status: {{ myForm.status }}   formGroup.status captures form validity.

formGroup.setValue() method sets value for the entire formGroup. A value needs to be supplied for every single form control.


formGroup.patchValue() method sets value for a subset of formControls. This example patches firstName, lastName and city.


'| json' pipe is used to display 'myForm.value' as a JSON object.

You have entered: {{ myForm.value | json }}


FormGroup.get() method allows access to FormControl or sub FormGroup. Remember to type cast as FormControl or FormGroup, since get() method returns a generic AbstractControl object.
Your full name is {{ firstName.value }} {{ lastName.value }}
