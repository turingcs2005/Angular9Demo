Material table
Material tables can be very fancy, with pagination, filtering and sorting utilities. Below are steps to implement pagination:
  • Import MatPaginator and MatTableDataSource classes into component. Set [pageSize] in mat-paginator.
  • Implement mat-table and mat-paginator in template.
  • Link dataSource and paginator in component.
    • Use ViewChild() to access mat-paginator. @ViewChild(MatPaginator, {tatic: true}) paginator: MatPaginator; Note that the second argument needs to be true.
    • Create a MatTableDataSource instance. (Standard HTML5 tables do not have paginator property.)
    • Assign paginator to table in onInit(): this.dataSource.paginator = this.paginator
    • In .scss, use {display: flex} on mat-paginator component, so it is left-aligned.
